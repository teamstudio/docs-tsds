# 使われているライブラリを開く

使われているライブラリを開くときは、再設計が必要な可能性があるライブラリを探す方が簡単です。

データベース設計の引き継ぎ元のライブラリをすべて開くには、右側のペインでデータベースを少なくとも 1 つ選択し、**[ データベース ]** メニューから **[ 使われているライブラリを開く ]** を選択します。検索するサーバーを選択するプロンプトが表示されます。Design Manager の左側のペインに、一致するテンプレートの一覧が表示されます。

## 特定のデータベース設計によって使用されているすべてのライブラリを特定して開くには
1. 右側のペインでデータベースを少なくとも1つ選択します。
2. **[データベース]** メニューから、**[使われているライブラリを開く]** を選択します。
3. **[ サーバー ]** ドロップダウンから、検索するサーバーを選択して **[OK]** をクリックします。
