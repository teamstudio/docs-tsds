    リリースノート

    Edition 34 は Teamstudio Notes Tools の機能リリースです。シリアル番号とキーを管理する新しい UI に加えて、このリリースでは最新版の Notes と Windows に対応するための数多くの内部的なアップグレードが組み込まれています。

    新機能のキー管理 UI の詳細はTeamstudio Tools の初期化を参照してください。
